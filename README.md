# config

自用配置和脚本

## windows env setup

### install scoop

install [Scoop](https://scoop.sh/#/) first.

```sh
scoop bucket add extras
scoop bucket add nerd-fonts
scoop bucket add java
```

```sh
scoop install uutils-coreutils
scoop install curl wget
```
